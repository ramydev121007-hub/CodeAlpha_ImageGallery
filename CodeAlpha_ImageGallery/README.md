# CodeAlpha Internship Project: Interactive Image Gallery

## Project Overview
This is a responsive, interactive image gallery built with HTML, CSS, and JavaScript as part of the CodeAlpha internship program. The project demonstrates modern web development techniques including responsive design, DOM manipulation, and interactive UI components.

## Live Demo
[View Live Project](https://yourusername.github.io/CodeAlpha_ImageGallery/)

## Features Implemented

### ✅ Core Requirements:
1. **HTML & CSS Layout**: Responsive grid-based image gallery
2. **JavaScript Navigation**: 
   - Next/Previous buttons for gallery navigation
   - Lightbox view for full-screen image display
3. **Hover Effects & Transitions**:
   - Smooth image scaling on hover
   - Caption slide-up animation
   - Button hover effects with 3D transforms
4. **Responsive Design**:
   - Adapts to mobile, tablet, and desktop screens
   - Flexible grid layout with media queries

### ✅ Bonus Features:
5. **Image Filters**: 
   - Grayscale, Sepia, Brightness, Contrast, Hue-rotate, Saturate
   - Visual filter indicators on each image
6. **Category Filtering**:
   - Filter images by: All, Nature, Urban, Abstract, Portrait
   - Active state highlighting for filter buttons

### ✅ Additional Features:
7. **Keyboard Navigation**: Arrow keys for lightbox navigation, ESC to close
8. **Smooth Scrolling**: Navigation buttons scroll to images
9. **Performance**: Uses Unsplash CDN for optimized images
10. **Accessibility**: Proper ARIA labels and keyboard support

## Technologies Used
- **HTML5**: Semantic markup structure
- **CSS3**: 
  - Flexbox & Grid for layouts
  - CSS Variables for theming
  - Transitions & Animations
  - Media Queries for responsiveness
- **JavaScript (ES6+)**:
  - DOM manipulation
  - Event handling
  - Dynamic content rendering
- **Font Awesome**: Icon library
- **Unsplash API**: High-quality demo images

## How to Use
1. **Browse Images**: Scroll through the responsive grid gallery
2. **Filter Images**: Click category buttons to filter by type
3. **View Details**: Hover over images to see captions and filter types
4. **Lightbox View**: Click any image to open full-screen view
5. **Navigate**: Use on-screen buttons or keyboard arrows in lightbox
6. **Close Lightbox**: Click X, press ESC, or click outside the image
